More Like This

This Stremio addon shows similar movie/series recommendations gathered from multiple external sources. When you search for a movie or TV show in Stremio, the addon will show recommended movies/series in the addon's catalog.

You will need your own API Keys for the recommendation sources. Click the links below each input field to get your free API keys. You only need an API Key for the source that you want. For example, if you only want TMDB recommendations, you only need to provide a TMDB API key.

Optional Settings
Use TMDB over Cinemeta for metadata
?
By default, the addon uses Stremio's primary addon, Cinemeta, to get movie and series data. Toggle the setting to prioritize TMDB over Cinemeta. Requires a TMDB API key.
Enable Manual Title Searching
?
Enable to find recommendations from movie/show title searches as well. Note: Title-based searches may be less accurate than using the exact ID obtained from Stremio.
