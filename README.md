# More Like This

**More Like This** is a Stremio addon that fetches similar movie and TV show recommendations from multiple sources, all in one place.

# Features

- Pulls recommendations from four different sources:
  - **TMDB**
  - **Trakt**
  - **Gemini AI**
  - **Tastedive**
- Find recommendations by searching for:
  - **IMDb ID**
  - **Kitsu ID**
  - **Title**
- Supports **RPDB** posters

# Getting Started

## Requirements

- **API Keys**:  
  You’ll need API keys for the sources you want to use.  
  _(Example: If you only want TMDB recommendations, you only need a TMDB API key.)_  
  Links to get free API keys are provided in the addon’s configuration page.

## Installation

Install the addon directly from:  
➡️ [https://bbab4a35b833-more-like-this.baby-beamup.club/configure](https://bbab4a35b833-more-like-this.baby-beamup.club/configure)

Install and run locally

## Configuration Options

- **Combine Into One Catalog**:  
  By default, each recommendation source appears in its own catalog.  
  Enable this option if you want to merge all recommendations into a single catalog.

- **Set Catalog Order**:  
  Drag and drop to set the order of how recommendations catalog appears

- **Use TMDB over Cinemeta for metadata**:  
   By default, the addon uses Cinemeta (Stremio’s main addon) to fetch movie and show data.  
   Enable this setting if you prefer TMDB data instead. _(Requires a TMDB API key.)_

- **Title-Based Searching**:  
  By default, the addon will only work when searching for an IMDb ID or Kitsu ID.  
  Enable this to find recommendations based on movie or show titles searches as well.  
  _(Note: Searching by title may be less accurate than using the exact ID.)_

# Usage

Simply search for any IMDb ID, Kitsu ID, or movie/show title in Stremio.  
Similar movies and series will automatically appear in the addon’s catalogs.
