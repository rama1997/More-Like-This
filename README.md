# More Like This

**More Like This** is a Stremio addon that fetches movie and series recommendations from multiple sources

# Features

- Pulls recommendations from 4 different sources
  - TMDB
  - Trakt
  - Tastedive
  - Gemini AI
- Find recommendations by searching
  - IMDB ID
  - Kitsu ID
  - Title
- Supports RPDB posters

# Getting Started

## Requirements

- API Keys. You only need API Keys for the source that you want. For example, if you only want TMDB recommendations, you only need to provide a TMDB API key.

## Installation

Install directly from https://bbab4a35b833-more-like-this.baby-beamup.club/configure

## Configuration

- **Combine Into One Catalog**: By default, each recommendation source will have it's own catalog. Enable this setting if you want to merge all the recommendations into one catalog.
